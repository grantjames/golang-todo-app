About

You can call the API at the following endpoints or list all todos at /list.

  • GET /api/todos - Get all todos
  • GET /api/todos?overdue - Get overdue todos
  • GET /api/todos?status=[Completed|Started|Not Started] - Get todos by status
  • POST /api/todos - Create a new todo (JSON body: {"title": "Your title"}, "due": null)
  • GET /api/todos/{id} - Get a specific todo by ID
  • PUT /api/todos/{id} - Update a specific todo by ID (JSON body: {"title": "New title", "completed": true})
